# laravel-template

## Environment
- php7.2
- apache2
- mysql5.7
- phpmyadmin

## Getting Start
``` bash
// git clone
$ git clone https://github.com/kagepedia/laravel-template.git
$ cd laravel-template

// docker set up
$ docker-compose up -d

// laravel set up
$ cd app/laravel-template
$ touch .env
$ composer install
```

## Access
application  
http://localhost:80/

phpmyadmin  
http://localhost:8080/
